# Father’s Day for Mr. Owiah

A responsive, single-page Father’s Day tribute from Stephen, Shaviva, Kukua,
Ebo, and Eno Serwaa.

## Preview

Open `index.html` in a browser. The homepage gallery portal links to the complete
archive in `gallery.html`. No installation or build step is required.

## Family media

The photographs and videos in `assets` are used throughout the featured collage,
the swipeable memory reel, and the video gallery.
